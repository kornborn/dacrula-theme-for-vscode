# dacrula-theme-for-vscode

## Установка

- скопировать папку darcula-custom-go-theme в папку ~/.cursor/extensions или ~/.vscode/extensions
- перезапустить Cursor / VSCode
- выбрать Color Theme: Darcula Custom (Go Enhanced)

## Изменение

- все настройки цветов хранятся в darcula-custom-go-theme/themes/dackula-custom-go-theme.json
- можно попросить LLM поменять любой элемент на нужный цвет в этом файле
